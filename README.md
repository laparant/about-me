## Previous technical stack
For the past 6 years, I have driven the backend development of a core enterprise web platform, mostly written in Scala, deployed in AWS, and backed by a PostgreSQL database. I owned the lifecycle of our features from initial design and technology choices to actual implementation, testing, QA and deploying through CI/CD. For instance, I introduced ZIO and functional principles to the codebase, and I managed an iterative refactoring to standardize all the backend services, providing an easier onboarding to newcomers. I was also involved in some other technologies such as CubeJS, written in Typescript, or even Python to integrate with AI prototypes.
